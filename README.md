## 🇷🇺 Русский

### Описание

Набор конфигурационных файлов для Hyprland (Wayland compositor) с обоями и скриптом автоустановки.

### Содержимое

| Папка | Описание |
|-------|----------|
| `.config/hypr` | Основной конфиг Hyprland |
| `.config/waybar` | Панель задач (top bar) |
| `.config/rofi` | Приложение запуска (app launcher) |
| `.config/kitty` | Терминал |
| `.config/wofi` | Меню приложений |
| `.config/wlogout` | Экран блокировки |
| `.config/hyprlock` | Локер экрана |
| `.config/fastfetch` | Системный информация |
| `wallpapers/` | Обои (14 шт) |
| `install.sh` | Скрипт автоустановки |

### Установка

```bash
# Клонирование
git clone https://github.com/mark-smitov/AstraDothFiles.git
cd AstraDothFiles

# Запуск (без прав root - только symlinks)
./install.sh

# ИЛИ полная установка (с драйверами, от root)
sudo ./install.sh
```

### Требования

- Arch Linux / Manjaro
- Hyprland
- NVIDIA драйвер (опционально)

## 🇬🇧 English

### Description

Configuration files for Hyprland (Wayland compositor) with wallpapers and auto-install script.

### Contents

| Folder | Description |
|--------|-------------|
| `.config/hypr` | Hyprland main config |
| `.config/waybar` | Top bar / taskbar |
| `.config/rofi` | App launcher |
| `.config/kitty` | Terminal |
| `.config/wofi` | Application menu |
| `.config/wlogout` | Logout screen |
| `.config/hyprlock` | Screen lock |
| `.config/fastfetch` | System info |
| `wallpapers/` | Wallpapers (14 pcs) |
| `install.sh` | Auto-install script |

### Installation

```bash
# Clone
git clone https://github.com/mark-smitov/AstraDothFiles.git
cd AstraDothFiles

# Run (without root - symlinks only)
./install.sh

# OR full install (with drivers, as root)
sudo ./install.sh
```

### Requirements

- Arch Linux / Manjaro
- Hyprland
- NVIDIA driver (optional)
